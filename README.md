# 뱀 게임
## 구현 기능
  
1. 시작 화면 만들기(난이도 설정 가능한 시작화면) - 김경오
  + 난이도 설정(쉬움, 보통, 어려움)
  + 난이도에 따라 프레임 레이트 변경하기
  + 현재 난이도를 시작 화면에 표시
  + 게임 배경 수정
